# ProgrammingProjectII

## Installing

Dependencies: Docker

Linux:

`source ./setup_env.sh`

Windows:

Set the environment variables `GOOGLE_PROJECT_ID` and `GOOGLE_APPLICATION_CREDENTIALS` as defined in the `setup_env.sh` file.


## Running

`docker-compose up -d`

## Stopping

`docker-compose down`
